# Findings Summary

## File Analysis

The File Analysis tab was used to inspect the disk image file system. This view supported review of file paths, file types, sizes, and timestamp metadata. It helped identify files of interest before deeper review.

## Keyword Search

The Keyword Search tab was used to search for investigator-defined terms across the disk image. Search results reduced the amount of data requiring manual review and highlighted files that may contain relevant evidence.

## Timeline Analysis

The Timeline Analysis tab was used to reconstruct activity in chronological order. This is the strongest DFIR value of the Autopsy lab because it supports incident reconstruction: instead of only listing artifacts, the timeline helps explain when events occurred and how activity progressed.

## Reports

Reports were generated to preserve the investigation output:

- File Listing Report
- Keyword Search Report
- Timeline Report

## Limitations

The lab writeup documents the forensic workflow and report types. Screenshots are stored in the `screenshots/` folder, and the downloadable lab PDF is stored in the `reports/` folder.
